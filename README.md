# mkdocs-example

```sh
pip install \
    mkdocs \
    mkdocs-material \
    mkdocs-minify-plugin \
    mkdocs-autolinks-plugin \
    pymdown-extensions \
    mkdocs-git-revision-date-localized-plugin
```

Create the docs repository.

```sh
mkdocs new docs
mkdocs serve
mkdocs build
```
